# Networking Labs | Day 1

## Objective

Understand the fundamentals of networking and become familiar with basic networking commands in Windows.

## Topics Covered

- Computer Networks
- LAN vs WAN
- IP Addressing
- DNS
- Routers
- Default Gateway
- Network Adapters
- HTTP vs HTTPS

## Commands Used

```bash
ipconfig
ping google.com
ping ecampus.fuoye.edu.ng
tracert ecampus.fuoye.edu.ng
getmac
```

## Key Takeaways

- Every device on a network has an IP address.
- DNS translates domain names into IP addresses.
- A failed ping does not always mean a server is offline.
- Traceroute helps identify the path packets take across networks.
- Networking troubleshooting requires gathering evidence before drawing conclusions.

## Reflection

Today's lab helped me connect networking theory with practical experience. Running the commands myself and analyzing their output gave me a much better understanding of how devices communicate over a network.
